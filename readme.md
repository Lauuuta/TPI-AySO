**EJECUTAR main.py**

Este es un porgrama interactivo que permite administrar y consultar las calificaciones de un listado de estudiantes. Cuenta con un menú interactivo, control de errores para entradas de usuario y búsquedas insensibles a mayúsculas/minúsculas (*case-insensitive*).

el menu cuenta con las suiguientes caracteristicas 
-Muestra un listado completo de todos los estudiantes y sus respectivas notas.
-Calcula el promedio de un estudiante y determina si está o desaprobado .
-Muestra de forma exacta la nota final promedio de cada alumno.
-Filtra los nombres sin importar si se escriben en mayúsculas, minúsculas o una combinación de ambas
-Sistema robusto ante errores de entrada utilizando bloques `try-except` para evitar que el programa se cierre inesperadamente.
